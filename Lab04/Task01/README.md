# Пояснення коду: Task 01 (Дек на основі масиву)

Цей код реалізує **Дек** (двобічну чергу) за допомогою кільцевого масиву. Це означає, що коли індекс доходить до кінця масиву, він "перестрибує" на початок за рахунок операції залишку від ділення `%`.

## Розбір класу `Deque`:

```csharp
public sealed class Deque
```
`sealed` означає, що від цього класу не можна успадковуватися.

```csharp
private readonly int[] _buffer; // Сам масив, де лежать дані
private int _front;             // Індекс "голови" (початку)
private int _back;              // Індекс "хвоста" (кінця)
```
Поля класу. `_front` вказує, куди вставляти новий елемент на початок. `_back` вказує куди вставляти в кінець.

```csharp
public Deque(int capacity)
{
    _buffer = new int[capacity];  // Виділяємо пам'ять під масив заданого розміру
    _front = 0;                   // Початкова точка для голови
    _back = capacity - 1;         // Початкова точка для хвоста (щоб вони не перетнулись на старті)
    Count = 0;                    // Спочатку дек порожній
}
```
Конструктор ініціалізує буфер. Ми ставимо голову і хвіст на різні кінці так, щоб при першому додаванні вони зустрілись.

```csharp
public bool PushFront(int value)
{
    if (IsFull) return false;        // Якщо місця немає — виходимо
    _front = PreviousIndex(_front);  // Крок 1: Зсуваємо голову "вліво" (по колу)
    _buffer[_front] = value;         // Крок 2: Записуємо туди наше число
    Count++;                         // Крок 3: Збільшуємо загальну кількість
    return true;
}
```
Додавання на початок. Чому спочатку зсуваємо `_front`? Бо поточний `_front` може вже зберігати якесь значення. Ми посуваємо індекс вліво.

```csharp
public bool PushBack(int value)
{
    if (IsFull) return false;      // Перевірка переповнення
    _back = NextIndex(_back);      // Крок 1: Зсуваємо хвіст "вправо" (по колу)
    _buffer[_back] = value;        // Крок 2: Записуємо значення
    Count++;                       // Крок 3: Збільшуємо лічильник
    return true;
}
```
Те саме, тільки в кінець. Хвіст рухається "вперед".

```csharp
public bool PopFront(out int value)
{
    if (IsEmpty) { value = default; return false; } // Якщо порожньо - нічого діставати
    value = _buffer[_front];         // Крок 1: Читаємо значення з поточної голови
    _front = NextIndex(_front);      // Крок 2: Зсуваємо голову "вправо" (відрізаємо її)
    Count--;                         // Крок 3: Зменшуємо лічильник
    return true;
}
```
Забираємо з початку. Запам'ятали значення, посунули голову вперед (що фактично робить наступний елемент новою головою).

```csharp
private int NextIndex(int index) => (index + 1) % Capacity;
```
Метод для руху "вправо". Залишок від ділення `% Capacity` гарантує, що якщо `index + 1 == Capacity`, ми отримаємо `0` (перескочимо на початок).

```csharp
private int PreviousIndex(int index) => (index - 1 + Capacity) % Capacity;
```
Метод для руху "вліво". Додаємо `Capacity` перед взяттям залишку, щоб уникнути від'ємних чисел (якщо `index == 0`, то `0 - 1 = -1`, а `-1 + Capacity` дасть останній індекс масиву).
