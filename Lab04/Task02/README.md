# Пояснення коду: Task 02 (Черга за допомогою 2-х стеків)

Цей код моделює роботу **черги** (перший прийшов — перший вийшов) за допомогою двох **стеків** (останній прийшов — перший вийшов), які вбудовані в C#.

## Розбір класу `QueueTwoStacks`:

```csharp
private readonly Stack<T> _inStack;
private readonly Stack<T> _outStack;
```
Ми маємо два стеки. `_inStack` — приймає всі нові елементи. `_outStack` — віддає елементи, коли ми робимо діставання (Dequeue).

```csharp
public void Enqueue(T item)
{
    _inStack.Push(item);
}
```
Метод для додавання. Він максимально простий — ми просто "закидаємо" число в перший стек `_inStack`. Складність **O(1)**.

```csharp
public T Dequeue()
{
    if (IsEmpty) throw new InvalidOperationException("Queue is empty");
    MoveInToOutIfNeeded();
    return _outStack.Pop();
}
```
Метод для видалення (вилучення) першого числа в черзі. Оскільки в нас елементи лежать в `_inStack` "навпаки", ми спочатку викликаємо допоміжний метод `MoveInToOutIfNeeded()`, який переверне цей стек у нормальну чергу. А потім просто беремо елемент з "перевернутого" стеку `_outStack`.

```csharp
private void MoveInToOutIfNeeded()
{
    if (_outStack.Count == 0)
    {
        while (_inStack.Count > 0)
        {
            _outStack.Push(_inStack.Pop());
        }
    }
}
```
Головний метод, який робить всю "магію". Він перевіряє: якщо стек `_outStack` (з якого ми дістаємо елементи) пустий, то він бере **всі** елементи з `_inStack` і по одному перекидає їх в `_outStack`.
Наприклад, якщо в `_inStack` лежало `1, 2, 3` (де 3 на горі), ми перекинемо в `_outStack` спочатку 3, потім 2, потім 1. Тепер в `_outStack` лежить `3, 2, 1` (де 1 на горі). В результаті найстаріший елемент опинився зверху і готовий до вилучення! Цей метод працює за **O(N)**, але викликається рідко.
